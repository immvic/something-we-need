# Victor Chironachi - Portfolio

This is a modern, minimalist portfolio site for **Victor Chironachi**, an aspiring engineer with a background in Robotics and 3D Design.

## 🛠️ Features
- **Sleek Black-and-White Aesthetic:** High-contrast design for maximum readability.
- **React & TypeScript:** Modern tech stack for high performance.
- **Vite:** Extremely fast development and build times.
- **Mobile Responsive:** Designed to look great on any device.

## 🚀 Getting Started

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
To run the project locally in development mode:
```bash
npm run dev
```

### Build
To create a production-ready build in the `dist/` folder:
```bash
npm run build
```
